# Workspace folder

This folder can be used to work with projects that are not part of the repository, but you want to use the claude code.

For example, you can create a new project in this folder and use the claude code to run it. You can also use this folder to test new features or to work on new projects without affecting the main repository.

## How to use

1. Create a new folder in the `workspace` folder for your project.
2. Clone a new repository in that folder or create new files for your project.
3. open claude in that folder and start working on your project.
